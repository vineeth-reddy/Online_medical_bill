============ Instruction ============
1. Create a database (for example, codexworld) and import the "cart_db.sql" file into this database.
2. Open the "dbConfig.php" file => change the $dbHost, $dbUsername, $dbPassword, and $dbName variable's value as per your phpMyAdmin details.
3. Browse the "index.php" file on the browser and check the PHP shopping cart functionality.
